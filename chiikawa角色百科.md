# Chiikawa 角色百科

> 基于 [chiikawati](https://github.com/txdywy/chiikawati) 项目整理，共 22 张正式角色卡。

---

## 性格 Trait 系统说明

全角色共用一套 16 维 trait 系统，用于描述性格倾向：

| Trait | 含义 |
|-------|------|
| gentle | 温柔敏感 |
| support | 接人补位 |
| chaos | 冲动混乱 |
| drama | 戏感与存在感 |
| chill | 松弛与留白 |
| leader | 压场与推进 |
| sunny | 热场与外放 |
| quiet | 安静陪伴 |
| orderly | 流程与规矩 |
| service | 待客与照顾 |
| menace | 危险感与攻击性 |
| odd | 怪异与反转 |
| wish | 稀有感与愿望感 |
| perform | 营业与舞台感 |
| huge | 体量感与强度感 |
| magic | 邪门点子与魔性 |
