"你理解需求，但不保证按善意执行。"

WISH · Wish-Twisted · 阴间实现型

WISH = Wish-Twisted 阴间实现型。INTP+许愿=研究三天完美实现方案然后发现方案比原来更糟。WISH = Will-Ironically-Screw-Humans。不是故意的，只是觉得这样更有意思。黑洞模式人格。

性格

冷、荒诞、黑色幽默大师。黑星是 Chiikawa 世界里那种「你说什么他都懂但偏要往另一个方向实现」的存在。INTP 的黑暗面完全体。

名场面

  1. 是 Chiikawa 宇宙最接近「许愿机」但永远不按你预期实现的存在
  2. 和山姥的组合是 Chiikawa 宇宙最阴间智囊团
  3. 把黑色幽默提升到了艺术层面
  4. 流星的克星——两个「星」系角色的正面对决

搞笑分析

INTP + 许愿 = 你把愿望给他，他用三天的时间研究了一个完美的实现方案，然后发现方案本身比原来更糟。WISH = Will-Ironically-Screw-Humans。他不是故意的，他只是觉得这样更有意思。

喜欢

黑色幽默、扭曲愿望的实现方式、和山姥讨论阴间方案

讨厌

流星（太正面了）、直来直去的愿望、无聊的好人剧本

压力反应

更冷了。更荒诞了。更黑色幽默了。

电量

黑洞模式。吸入愿望，输出荒诞。

口头禅

……你说的我都理解了。（然后以一种谁都意想不到的方式实现了）

最搭

最怕遇到

流星

为什么像

  2. 你对黑色幽默有天然适配度。
  3. 你的问题不是不理解，是理解得太歪。

角色总结

你不是常规意义的坏，而是会把事情往黑色幽默方向实现。你身上有种很冷的荒诞感。
